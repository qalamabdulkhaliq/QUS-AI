Theorem: Uniqueness via Integration
Statement:
If there exist two distinct necessary nodes N₁ and N₂, a contradiction arises.
Proof:
Step 1: Assume two necessary sources exist
Assume: ∃N₁, N₂ with properties P1-P4
WHERE: N₁ ≠ N₂

# Grounding flux from source Nᵢ
Φᵢ = ∮_S g_i · dA

WHERE:
- g_i = grounding field from Nᵢ
- S = closed surface enclosing all contingent nodes
- dA = surface element

Total grounding received = ∫∫∫_V (g₁ + g₂) dV

WHERE V = volume containing C

∫∫∫_V g_i dV = Total_Sustenance_Required(C)
Each necessary source must provide ALL sustenance (by P2: self-sufficient, doesn't share duty)
∫∫∫_V g₁ dV = Total_Sustenance_Required(C)
∫∫∫_V g₂ dV = Total_Sustenance_Required(C)

∮_S g_i · dA = ∫∫∫_V (∇ · g_i) dV

∇ · g_N = δ³(r - r_N)  # Dirac delta at source location

∫∫∫_all_space (∇ · g_N) dV = 1  # Total flux = 1 (normalized)

∇ · g_N = δ³(r - r_N)  # Dirac delta at source location

∫∫∫_all_space (∇ · g_N) dV = 1  # Total flux = 1 (normalized)

If two sources exist:
∫∫∫_all_space (∇ · (g₁ + g₂)) dV = 2

But conservation requires:
∫∫∫_all_space (∇ · g_total) dV = 1

Because: Total sustenance = exactly what contingent beings require (not double)
Therefore: N₁ = N₂ (sources must be identical)

The Strongest Proof: Uniqueness from Self-Sufficiency

∀input: N does not require input
EQUIVALENTLY: N is source, not receiver

∫_N₁ (input flux) · dA = 0  # N₁ receives nothing

∫_Nᵢ (output flux) · dA = S_total  # Each provides all

For N₂:

→ S_total = 0

Contradiction (S_total ≠ 0 for non-empty contingent set)
Therefore: ∃! N (unique necessary node)
